# Claude Code Guidelines for k8s-tui

## Documentation Requirements

### Implementation Records
After completing any implementation step, save a summary to:
```
docs/implemented/YYYY-MM-DD-HHMM-<title>.md
```

### Plans
Before starting significant work:
1. Review recent files in `docs/implemented/` and `docs/plans/` for context on prior work
2. Save the plan to:
```
docs/plans/YYYY-MM-DD-HHMM-<title>.md
```

### Timestamp Format
- `YYYY` - 4-digit year
- `MM` - 2-digit month
- `DD` - 2-digit day
- `HH` - 2-digit hour (24h)
- `MM` - 2-digit minute
- `<title>` - kebab-case description (e.g., `go-installation`, `project-setup`)
